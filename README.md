# Scripts
### Удаление комментариев из файла
```bash
./no-comments.sh <имя_файла>
```

### Скачивание сегментов видео с slurm.io
```bash
./slurm-video-download.sh
```

### Создание SSL сертификата
```bash
./cert-gen.sh name.example.com
```

### Извлечение сертификатов
```bash
./cert-ext.sh name.kubernetes.local
```
